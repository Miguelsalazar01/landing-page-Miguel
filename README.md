# PetAdote

## Descrição

PetAdote é uma landing page simples que apresenta uma ideia de aplicação web para ajudar na adoção de animais abandonados.  
A plataforma conecta abrigos e pessoas interessadas em adotar cães e gatos, facilitando que esses animais encontrem um novo lar.

O objetivo é conscientizar sobre o abandono de animais e incentivar a adoção responsável.

## ODS Relacionada

Este projeto se relaciona com a **ODS 15 – Vida Terrestre**, que busca proteger a vida animal e promover o cuidado com os ecossistemas terrestres.

A iniciativa contribui ao incentivar a adoção de animais e reduzir o abandono.

## Tecnologias Utilizadas

- **HTML5** — estrutura da página
- **CSS3** — estilização e layout da landing page
